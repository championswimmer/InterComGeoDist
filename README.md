# Geo Distance Invitation List

## Requirements

1. Java (JDK 8 or above)
2. Kotlin (not needed to install separately)

## Compile and Run

Compile project

```shell
./gradlew jar
```

Run Program

```shell
java -jar build/libs/IntercomGeoDist-1.0-SNAPSHOT.jar
```

## Run Tests

```shell
./gradlew check 
```
